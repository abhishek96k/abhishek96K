# 🔥 Featured Project

## 🏋️ Gymatic 
[![Live Demo](https://img.shields.io/badge/Live-Demo-green?style=for-the-badge&logo=vercel)](https://gymatic.tech)

🚀 Live: https://gymatic.tech
  
A gym management SaaS designed for gym owners to manage:
- members
- payments
- dues
- renewals

### Key Highlights:
- Built with **Next.js, React, TypeScript, MongoDB**
- PWA support for mobile-friendly usage
- Renewal tracking with hot/warm/cold member segmentation
- Dashboard for revenue, members, and payment insights
- Designed for simple real-world business operations
